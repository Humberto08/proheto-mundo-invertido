
# Projeto Mundo Invertido

Projeto construído para fins didáticos, com o objetivo de colocar em prática os conhecimentos de HTML, CSS e JavaScript aprendidos na DIO


## Stack utilizada

**Front-end:** 
 - HTML5 
 - CSS3 
 - JavaScript

 **Back-end:**
 - JavaScript
 - NodeJS
 - Firebase
